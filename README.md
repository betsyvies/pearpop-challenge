## Requirements

- `node` v16.13.0
- `npm` v8.1.0

## Setup

1. Clone the repository.

```sh
git clone git@github.com:betsyvies/pearpop-challenge.git
```

2. `cd` into the cloned repository.

```sh
cd pearpop-challenge
```

3. Install the dependencies.

```sh
npm install
```
## Running locally

```sh
npm run dev
```
